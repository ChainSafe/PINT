type CallFilter: Contains<<Self as Config<I>>::Call>

Pre-dispatch filter for incoming calls.

The pallet will filter all incoming calls right before they’re dispatched. If this filter rejects the call, special event (Event::MessageCallRejected) is emitted.

[src]

type EncodedCall: Decode + Encode + Into<Result<<Self as Config<I>>::Call, ()>>

The type that is used to wrap the Self::Call when it is moved over bridge.

The idea behind this is to avoid Call conversion/decoding until we’ll be sure that all other stuff (like spec_version) is OK. If we would try to decode Call which has been encoded using previous spec_version, then we might end up with decoding error, instead of MessageVersionSpecMismatch.
