Crate pallet_bridge_dispatch[−][src]

Expand description

Runtime module which takes care of dispatching messages received over the bridge.

The messages are interpreted directly as runtime Call. We attempt to decode them and then dispatch as usual. To prevent compatibility issues, the Calls have to include a spec_version. This will be checked before dispatch. In the case of a successful dispatch an event is emitted.
