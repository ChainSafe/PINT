Module bridge_runtime_common::messages[−][src]

Expand description

Types that allow runtime to act as a source/target endpoint of message lanes.

Messages are assumed to be encoded Calls of the target chain. Call-dispatch pallet is used to dispatch incoming messages. Message identified by a tuple of to elements - message lane id and message nonce.
